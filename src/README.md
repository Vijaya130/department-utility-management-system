## Department Utility Management System

A web-based application built using React and Supabase to streamline and manage various department utilities such as mentor-mentee allocation, student data, and academic tracking.

---

## Features

- 👤 Role-based access (Admin / Mentor)
- 📋 View and manage mentor-mentee data
- 🎓 Categorization of students by year (2nd, 3rd, 4th)
- 🔍 Search functionality for students and faculty
- 📊 Display student details (USN, Name, Mentor)
- ⚡ Dynamic UI updates
- 🌐 Real-time data handling using Supabase

---

## Tech Stack

- Frontend: React (Vite)
- Backend/Database: Supabase
- Styling: CSS
- Version Control: Git & GitHub

## Installation & Setup

1. Clone the repository:

git clone https://github.com/Vijaya130/department-utility-management-system.git

2. Navigate to the project folder:

cd department-system

3. Install dependencies:

npm install

4. Run the project:

npm run dev

---

## Supabase Integration

- Supabase is used as the backend database.
- Data is fetched and displayed dynamically using API calls.
- Configure your Supabase URL and API key in "supabase.js".
